富爸爸穷爸爸的读书笔记

十月 6, 2018 发布在 reading notes, financial quotient

18年下半年，动荡变化；工作上忙于日常的填坑和撸码，疲惫不堪，不想陷入“熟练工”的陷阱，于是趁着假期打算思考一下接下来的规划。偶然间走过福州路看到书摊上的“富爸爸穷爸爸”，印象中这本书当时的争议很大，讲的内容围绕两种截然不同的财务思维，作为假期第一本书，可以说是久违的平静中，涌起了一股上升的力量。

第一课 : 富人不为钱工作

一、 故事从罗伯特(作者)9岁时开始，当时和他的小伙伴迈克因为被拒绝邀请到同学家的别墅独家，被看作是“穷孩子”而萌发了赚钱的念头。最初尝试的竟然时利用牙膏皮里的铅加上模具，重新铸造出硬币。虽然出发点和动手创新能力是值得夸赞的，不过这个方案却不是一个好方法，在被罗伯特的父亲教训之后，他们得到的建议是和迈克的父亲谈一谈，因为作为一个八年级都没念完的大人，却经营着多家公司。

富爸爸设计通过一门和真实生活相近的课程来推动他们学习关于挣钱的知识。而课程内容则是每周六在富爸爸的便利店工作三小时，作为回报，他们会获得30美分的报酬。 很快，罗伯特就对枯燥的工作和可怜的薪水感到了厌倦，并在其他人的怂恿下，认为富爸爸是在剥削廉价的劳动力，而非真正教授他们赚钱的知识。

注：这里引出如下了问题

在生活中，人们往往是被推着向前进。有的人在遇到挫折或者问题时，会变得愤怒，将问题推到老板或身边人的身上；而有的人则从中汲取教训，并以此不断学习进步。那我们该如何解决问题？

二、 富爸爸设计的课程就是希望当罗伯特和迈克在感到愤怒时，学会转变，开动头脑的思维能力，保持学习的激情，从而了解如何让钱为自己工作。

  • 穷人和中产为钱工作，而富人让钱为自己工作？。（税收所扮演的角色）
  • 避开一生中最大的陷阱 - 老鼠陷阱。 （被恐惧和贪婪支配）
  • 学会让感情跟随思想，而不是让思想跟随感觉 （控制贪婪的欲望）

三、 具体方法：忘记工资，继续使用头脑思考，不求回报地工作，不断观察身边的机会，最终会找到比拿工资更赚钱的方法。 罗伯特和迈克在杂货店继续工作着，偶尔间发现了一个利用废弃卖不出去的图书开办阅读室的生意，这样一来，他们在近三个月中平均每月能赚到9.5美元，在当时对于孩子而言也已经是非常客观的报酬了。尽管最终因为斗殴问题，阅读室关闭了，但是罗伯特和迈克第一次接触并了解了如何让钱为他们工作。

第二课： 为什么要教授财务知识

一、通过一些真实的例子引出——“即便是有钱人，也会在某些时候因为缺乏财务知识，而变得一无所有”。 所以重要的不是你挣了多少钱，而是你能留下多少钱，以及能留多久。

规则 明白资产和负债的区别，并且购买资产

富人获得资产，穷人和中产获得负债，只不过他们以为那些负债就是资本。

（我的理解：这里的是指在通过负债的方式去购买消费品，比如贷款买车，贷款买最新的Iphone等。 而贷款买房，于我而言，并不是简单满足自己的美好生活。 经历了2015～2016年的疯狂，到了2018年年底时，恰逢美国又发起了贸易战，所以整个国内经济应该实在一个调整和对抗的阶段。如果把握机会，或许在19年或者20年，能够以优惠的价格买入优质的资产。比如房价、比如从高位跌下来的腾讯、阿里和小米的股票等）

二、想学习如何致富，从哪里开始致富，就需要先学习财务知识（学校只教授知识，以便找一份好工作，可以赚钱养家，但这并不是真正的财务知识）。

区别 资产是无论我工作与否都能够把钱放进口袋的东西，而负债是把钱从口袋那出去的东西

观察穷人、中产和富人在资产负债表中现金流所体现出对于金钱的处理方式。

三、尽可能买入能带来收入的资产，并坚持下去； 通过控制支出和负债，在投资的同时获取经验后逐步成为“成熟的投资者”。

老鼠陷阱 努力的工作 首先得利的是公司、国家和银行，缴完各种税和贷款后的钱才能真正为自己所用

（我的理解： 努力的工作，带来的收入增长是伴随着税收的增长，往往可支配的金额小于劳动付出所得。 同时随着负债和支出项的增加，比如车贷、房贷等后，真正能够用来价值投资，包括个人成长的投资其实已经不多了。 但是如果反过来，资产所得，尤其在当下中国其实是不怎么缴税的。最直观的例子就是活期、定期以及其他一些投资收入）

第三课： 关注自己的事业

一、 为了财务安全，需要关注自己的事业（而非职业）。这里作者提到了一句“事业的重心是你的资产项”。

依附于工作，则意味着一旦失去工作收入，则会陷入困境。购买的车子、手表，高尔夫球杆等虽然在很多会计和银行被认定为资产，但实际出售的时候，获得的现金会小于纸面上的估值，同时还要面临税收。

二、 对于成年人而言，把支出保持在低水平、减少借款并勤劳工作会打下一个稳固的资产基础。

  • 不需要到场就可以正常运行的业务。拥有他们，但由别人管理。相反，如果需要我在那里工作，那就不是事业而成了职业了。
  • 股票
  • 债券
  • 能够产生收入的房地产（现金流）
  • 票据
  • 版税
  • 有价值、可产生收入或者增值潜力且流动性高的东西

三、 思考并找到你喜欢的资产，努力工作，把收入投入资产后，因为钱可以成为你的雇员，并24小时不断为你赚钱。

将享乐放在后面，是真正富人和穷人的区别。

第四课： 税收的历史和公司的力量

一、 政府机构的扩张，社会福利的上升会导致对于税收的胃口越来越大，最终无法避免地将税收范围扩大到中产和穷人阶级。而富人利用知识，寻找合理的避税方法和空间。

二、 学会雇佣聪明的会计/税务师，还有律师，支付一部分金钱给他们，会比交给政府的要少很多。

三、 财商由4部分组成： 财务知识、投资策略、了解市场供需和法律。

延展阅读： 在中国如何开公司，并去发现开公司带来的好处与保护。

第五课： 富人的投资

一、 在现实生活中，人们往往是靠勇气而非智慧去取得领先的地位

(这里牵扯到一个教育的基本，985是培养科学家的。特别是中国的教育从某方面来说是惩罚失败的，而实际上，经历失败，也正是面对自己不足之处并学习提升自己的时候。 这就造成一种现象，就是985执行力强，但是对于55开的选择，往往无法快速决断，因为一直以来读书、考试都不允许失败；成绩中游的学生，带着半蒙半懂，结合时代的大机遇，做永远比不做好。)

二、 这是一个波澜壮阔的时代，需要抛弃陈旧的观念，接受信息，学习关于投资和财富的知识。

(需要了解不同的理财选项有着不同的意义，学习创新，对数字有敏感性。 这里结合立松叔的建议，通过读财报的方式，来了解一家公司的发展和潜力，从而进行投资。)

三、 再次强调，财商的4个重要组成部分： 1. 会计：即财务知识或者说是阅读数字的能力。 2. 投资：即以钱生钱的科学。 3. 了解市场：供给与需求。 4. 法律：必须按规则进行游戏。

可参考的投资工具：房地产作为基础投资，日复一日的积累，不断提供现金流，偶尔有价值的飙升；等待小型公司的股票快速增值。

四、 努力成为第二类投资者，即自己创造机会的投资者。

区别于第一类依赖中介机构，购买对应产品的投资人；

需要提高的技能：

  1. 寻找其他人都忽视的机会
  2. 增加自己
  3. 把精明的人组织起来

第六课： 学会不为钱工作

一、 对于穷爸爸而言，工作的稳定就是一切；对于富爸爸而言，不断学习才是一切。

二、 自身的培养，要避免尽在单一领域的专业技能被时代或者行业淘汰。有时候通过接触不同的岗位，去摸索商业体系的运行法则。

三、 成功所需要的技能包括：（1）现金流管理， （2）系统管理， （3）人员管理。

此外还需要一些销售和营销的技能，当然口头沟通能里也很重要。 既要成为好学生，也要努力成为好老师； 想要真正的富有，既要不吝付出，也要学会索取。

第七课： 客服困难

一、 仅拥有财务知识，有时还不能积累丰富的且能带来大量现金流的资产项，其主要的原因有5个： 1. 恐惧 2. 愤世嫉俗(杞人忧天，我个人觉得这么翻译好像更符合语境) 3. 懒惰 4. 不良习惯 5. 自负

二、 应对之法 1. 失败可以激励胜利者，而击败失败者 2. 不要被身边那些“嘈杂”的声音给影响，那些人他们本身并不是专业人士；通过分析看到那些怀疑者看不到的东西，发现被忽视的机会，而发现机会的能力正是关键。 3. 多想“我如何才能付得起”，而不是“我可买不起”。 需要多一点“贪婪”，多一点渴望拥有的东西。 4. 优先投资自己，支付自己需要的金钱，而后再支付债务。 以此来督促自己学得更多，投资得更多，拥有的更多，变得更强。财务上和精神上都便强。 5. 克服自负，发现在某一领域或者问题上有欠缺，就应该找领域的专家或者书，学习并提升自己。

第八课： 开始行动

  1. 找一个超现实的理由-精神的力量。 比如“不想一直打工，而想创造财富，拥有自己的产业。”

  2. 每天作出选择-选择的力量。 同时听新旧，正反的不同声音，从中找到自己的理解并作出选择。

  3. 慎重地选择朋友-关系的力量。 三人行必有我师，相同眼界的朋友，可以交换信息，互相学习对方的经验

  4. 掌握一种模式，然后再学一种新的模式-快速学习的力量 知识会落伍，关键在于快速学习新的知识，模式

  5. 首先支付自己-自律的力量 拥有胆量，不要软弱。 谨记两点：一是不要背上过大的债务包袱；二是遇到压力时不要轻易动用储蓄或资本，而是激发理财的天赋。

  6. 给经纪人以优厚的报酬-好建议的力量 懂得如何支付，并获得专业认识的建议，而不能只看眼前的小得失

  7. 做一个给予者-无私的力量 ``

  8. 用资产来购买奢侈品-专注的力量 ``

  9. 对英雄的崇拜-神话的力量 学习，模仿英雄的方式——成功的模式

  10. 先予后取-给予的力量 有舍才有得

第九课：还想要更多吗？还有一些其他的事

更多关于实操层面的指南

  1. 找一些做过你想做的事情的人
  2. 参加课程和研讨会
  3. 敢于提出低报价尝试交易
  4. 其他的可能还需要后面不断实践并深入理解
